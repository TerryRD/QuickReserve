# 👨‍🏫 助教 Staff 使用手冊

你是某位教練（Owner）旗下的助教。你能管理自己的時段與預約，但**看不到 Owner 或其他助教的資料**。

## 登入

教練 Owner 會給你一個邀請連結（`/invite/...`）。

1. 點開連結
2. 註冊（如果還沒帳號）
3. 接受邀請 → 進入後台 `/dashboard`

> **測試帳號**：`demo-staff-ming@example.com` / `Test1234!`（這位助教歸 林教練 旗下）

---

## 1. 你能做什麼

側邊選單：

| 項目 | 你能做 |
|------|--------|
| 儀表板 | 看你的統計 |
| 行事曆 | 新增 / 編輯 / 刪除**自己的**時段 |
| 預約管理 | 處理**屬於你時段**的預約 |
| 服務項目 | **唯讀**（你看得到，但不能建立或編輯） |
| 通知設定 | 個人通知偏好 |

> 你**看不到** Owner 或其他助教的時段與預約。

---

## 2. 行事曆操作

跟教練 Owner 一樣，可以：
- 新增單一時段
- 批量 / 重複規則
- 衝突偵測

但你**只能用既有的服務項目**（由 Owner 建立）。

如果服務不夠用、需要新增服務，請聯絡你的 Owner 教練。

---

## 3. 預約管理

學員預約你的時段後，會出現在 `/bookings` 中你能看到的範圍：
- **可確認 / 取消**：屬於你時段的預約
- 看不到：屬於 Owner 或其他助教時段的預約

---

## 4. 通知

跟 Owner 一樣：可在 `/settings/notifications` 啟用 Web Push + 設定偏好。

新預約申請會立刻通知到你的瀏覽器。

---

## 試試看（用測試資料）

> 測試帳號：`demo-staff-ming@example.com` / `Test1234!`

1. 登入 → 看左側側邊欄
   - 沒有「助教管理」選單（這是 Owner 限定）
2. `/calendar` → 看到你自己的時段（2 個）
   - **看不到** 林教練 的時段
3. `/services` → 看到 2 個服務（網球初級 / 進階） — 唯讀
4. `/bookings` → 看到屬於你時段的預約（若有）
