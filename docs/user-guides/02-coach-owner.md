# 🏓 教練 Owner 使用手冊

你是租戶的擁有者。你可以管理服務、行事曆、預約、助教。

## 登入

平台管理員會給你一個邀請連結（長得像 `quick-reserve-mu.vercel.app/invite/abc...`）。

1. 開那個連結
2. 若你還沒帳號 → 註冊（系統會自動把你帶回 invite 頁）
3. 若已有帳號 → 登入
4. 按「接受邀請」 → 進入你的後台 `/dashboard`

> **測試帳號**：`demo-coach-wang@example.com` / `Test1234!`

---

## 1. 儀表板（/dashboard）

進來看到：
- **公開連結**（你的 `/your-slug`）← 把這個傳給學員
- **2 個統計**：待確認預約數、本週時段數
- **2 個捷徑卡**：行事曆、服務管理
- **即將到來的預約** ← 最近 5 個

---

## 2. 服務項目（/services）

學員預約時必須先選一個「服務」。

### 新增服務

右上「+ 新增服務」 → 填：
- 名稱（必填，例如「桌球 1 對 1 課」）
- 描述（選填，告訴學員這堂課做什麼）
- 時長（分鐘，必填）
- 價格（可空 = 「洽詢」）

### 編輯 / 停用

每張服務卡片右下「編輯」打開原本表單修改。
停用後不會出現在公開頁，但既有預約保留。

> Staff 助教只能看不能改 — 服務由 Owner 統一管理。

---

## 3. 行事曆（/calendar）

### 看自己的時段

預設顯示本週，週一到週日。顏色：
- 🟦 藍 = 可預約
- 🟧 橘 = 待確認（學員預約後但你還沒按確認）
- 🟩 綠 = 已確認
- ⬜ 灰 = 已取消

點側邊的左右箭頭切換週次，「回本週」回今天。

### 新增單一時段

右上「+ 新增時段」 → 在 dialog 中選：
- 服務（從你建好的服務清單）
- 日期
- 開始 / 結束時間

按「新增」。
**衝突偵測**：如果跟既有時段重疊，會顯示「與以下時段衝突」並列出衝突的時段。

### 批量 + 重複規則（⚡ 批量/重複）

右上「⚡ 批量 / 重複」按鈕 → 彈出更大的 dialog：

1. **選服務**
2. **重複方式**：每天 / 每週 / 每月 / 每 N 天
3. **若選每週**：勾選星期幾（一三五、二四六等）
4. **若選每月**：選每月幾號
5. **開始日期 + 起迄時間**
6. **何時結束**：
   - `N 次後`（例如連續 12 週）
   - `截止日` 之前
   - `不限` ← 系統每天自動往前推 1 天，永遠維持 90 天可預約

底部會即時顯示「將建立約 N 個時段」。
按「建立」 → 若沒衝突直接建好；若有衝突會在 dialog 內列出衝突清單，你可以：
- 點每筆衝突右邊「跳轉去調整」→ 直接跳到那一週的行事曆
- 或點「略過衝突日，建立其他」 → 自動跳過衝突日、其他都建

### 多成員視圖（Owner 限定，FR-077）

如果你有助教，行事曆上方會出現「檢視」按鈕，點開有 checkbox 下拉清單，可以：
- **預設**：看自己 + 全部助教（slot 上會顯示成員名稱 badge）
- **取消勾選**特定助教 → 只看選的人
- **單獨看自己**：取消所有助教 → UI 變回「我的視圖」可以新增時段

其他成員的時段在你的視圖中為**虛線邊框 + 唯讀**，點開只能看不能改。

### 點擊行事曆上的時段（FR-080）

每個 slot 都可點擊 → 跳出 detail 視窗：
- 顯示：服務、時間、負責成員、狀態、學員（若已被預約）
- 若是**自己的可預約時段** → 顯示「刪除時段」按鈕
- 若是 `pending` / `booked` 狀態 → 顯示「前往預約管理」連結

### 管理重複規則（FR-073~076）

行事曆頁右上有「管理重複規則」按鈕 → `/calendar/rules`

每筆規則卡片顯示：
- 服務名稱 + 規則摘要（例「每週一三五 19:00–21:00 共 12 次」）
- 起始日期 + 結束條件
- **未來剩餘時段數**
- 右側：**啟用 / 停用** toggle + **刪除** 按鈕

**刪除規則時**會彈出對話框，可勾選：
- 「同時刪除未來尚未被預約的時段」（預設勾選）

⚠️ 已被學員預約（pending / booked）的時段**不會**被刪除，紀錄保留。

---

## 4. 預約管理（/bookings）

學員按下「預約」後，會出現在這裡，初始狀態 `待確認`（pending）。

### 篩選

頂部 5 顆按鈕：全部 / 待確認 / 已確認 / 已完成 / 已取消。

### 確認預約

每張「待確認」卡片右側有兩顆按鈕：
- **確認** → 預約變「已確認」、對應時段變綠
- **取消** → 預約變「已取消」、時段釋出回「可預約」

學員那邊會收到對應的 Web Push 通知（如果他啟用過）。

### 取消已確認預約

「已確認」狀態也能取消（教練有事不能上課）。一樣按「取消」即可。

---

## 5. 助教管理（/staff，僅 Owner）

### 邀請助教

填 email → 送出 → 拿到 invite URL → 傳給助教。

助教接受後會出現在下方列表，狀態 `已加入`。

### 移除助教

每位 staff 右側「移除」按鈕。這只是把 status 改成 `removed`（軟刪除），歷史資料保留。

---

## 6. 通知設定（/settings/notifications）

### 啟用 Web Push

進這頁第一次會看到「啟用瀏覽器推播通知」橫幅 → 點「啟用通知」 → 瀏覽器彈出權限詢問 → 同意。

之後新預約申請、學員取消等事件會即時推播。

### 偏好

可以開關：
- 每週日 20:00 收到下週預覽
- 每天早晨提醒今日行程（可調整時間 06:00-12:00）
- 預約前提醒（5 / 10 / 15 / 30 / 60 分 / 1 天前，可複選）
- 預約狀態變更立即通知

---

## 7. 學員（/customers，FR-070~072）

側邊「學員」進入，看到所有曾跟你的租戶預約過的客戶。

每張卡片顯示：
- 姓名 + 大頭貼
- **總預約數 / 待確認數 / 已完成數**
- 最近預約日期

> 不同教練的學員資料**完全隔離**（透過 `tenant_customers` bridge + RLS 強制）。

要看單一學員的所有預約細節，請到「預約管理」用 filter 篩選。

## 8. 公開連結

頂部 sidebar 底部有「🔗 公開預約頁 /your-slug」 → 點開複製 URL 給學員。

學員拿到後：
1. 開連結 → 看到你的介紹頁
2. 選服務 → 選日期 → 選時段 → 必須登入 / 註冊 → 送出預約
