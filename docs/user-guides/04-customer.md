# 👤 學員 / 客戶 使用手冊

你是要預約教練的人。

## 第一次預約

1. 教練給你一個專屬連結，例如：
   - `quick-reserve-mu.vercel.app/demo-wang-coach`
2. 開啟連結 → 看到教練介紹頁（hero 區會看到教練的描述與聯絡方式 — Email / 電話 / LINE，可直接點擊）

頁面上會有三步：

### Step 1 — 選擇服務

教練提供的所有服務（例如「桌球 1 對 1 課 60 分 $1500」），點一個。

### Step 2 — 選擇日期

**14 天** strip。每個日期下方顯示「N 個」可用時段。點要的日期。

想看更遠的日期？右上「再 7 天 ►」往後翻；「◄ 上週」回到上一段範圍。

### Step 3 — 選擇時段

當日所有可用時段，點一個 → 進入「確認預約」頁。

### 確認預約頁

顯示：日期、服務、時長、價格。

- 若你**未登入** → 系統跳轉到 `/login`，登入或註冊後自動回來
- 若你**已登入** → 可填備註（選填）→ 按「送出預約申請」

送出後狀態為**待確認**，教練確認後變「已確認」。

---

## 我的預約（/my-bookings）

登入後左上頭像可進去。會列出所有預約：

| 狀態 | 顏色 | 意義 |
|------|------|------|
| 待確認 | 🟧 黃 | 教練還沒按確認 |
| 已確認 | 🟩 綠 | 教練同意，正式成立 |
| 已完成 | ⬜ 灰 | 課程已上完 |
| 已取消 | ⬜ 灰刪除線 | 你或教練取消了 |

每張卡片若「課程時間還沒到」會有兩顆按鈕：
- **改時間**（FR-092~096）：跳轉到該教練公開頁，進入「改期模式」 → 選新時段 → 系統會原子性取消舊預約並建立新預約（同一個 transaction）。不可跨教練改期。
- **取消預約**：直接釋出時段。

---

## 啟用 Web Push 通知

進 `/my-bookings` 上方會看到「啟用瀏覽器推播通知」橫幅。

- 點「啟用通知」 → 瀏覽器跳出權限請求 → 同意
- 之後預約被確認 / 取消 / 改期會即時通知

**通知設定**：點右上「⚙️ 通知設定」可以選：
- 每週日 20:00 收到下週預覽
- 每天早晨提醒今日行程
- 預約前 N 分鐘提醒（30 分 / 1 小時 / 1 天前等）

## 試試看（用測試資料）

> 測試帳號：`demo-student-ming@example.com` / `Test1234!`

1. 開 `/login` → 用上面帳號登入
2. 自動進 `/my-bookings`：你會看到 2 筆預約
   - 王教練（待確認）
   - 林教練（已確認）
3. 開新分頁 `quick-reserve-mu.vercel.app/demo-chen-coach` → 選「高爾夫初學」→ 選一個有時段的日期 → 選時段 → 送出預約
4. 回 `/my-bookings` 應該看到新的一筆「待確認」
5. 點該預約的「取消預約」 → 即時消失（變已取消）

### 試試衝突偵測

開兩個 incognito 視窗：
1. 視窗 A：用 `demo-student-hua@example.com` 登入
2. 視窗 B：用 `demo-student-mei@example.com` 登入
3. 兩個視窗都打開 `/demo-wang-coach` → 都選**同一個**可用時段
4. 視窗 A 先按「送出預約」 → 成功
5. 視窗 B 也按「送出預約」 → 應該看到「該時段已不可預約」錯誤

這證明系統的原子鎖定有效，不會超賣。
