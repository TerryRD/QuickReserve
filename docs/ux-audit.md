# UX Audit — 各角色實際走過後的問題清單

> 從 4 個角色實際登入跑完整流程後整理。
> 標 ⭐ = 本次優先修；標 🟡 = 中期改善；標 ⚪ = 未來再說

## 🛠️ Platform Admin（你）

| # | 問題 | 嚴重度 |
|---|------|--------|
| P1 | 沒辦法快速進到某個教練的資料看（只能用 SQL 或公開頁） | ⭐ |
| P2 | 邀請過期了沒辦法重新邀請 / 看邀請紀錄 | 🟡 |
| P3 | 沒搜尋 / 排序 / 篩選租戶 | 🟡 |
| P4 | 想看「全平台所有預約 / 所有使用者」沒入口 | 🟡 |
| P5 | 無法幫某個教練重設密碼（只能去 Supabase Dashboard） | ⚪ |

## 🏓 教練 Owner

| # | 問題 | 嚴重度 |
|---|------|--------|
| O1 | **行事曆只能看一個成員** — Owner 想同時看全部助教 + 自己（你提出的） | ⭐ |
| O2 | **重複規則建完就消失** — 無法看 / 編輯 / 刪除既有規則，整批 slot 也沒法統一刪 | ⭐ |
| O3 | **行事曆上的時段點不開** — 看不到誰預約了、沒辦法刪除單一 slot | ⭐ |
| O4 | 沒「我的學員」清單 — 不知道有哪些客戶 / 他們的歷史 | ⭐ |
| O5 | 沒辦法 reschedule（只能取消後叫學員重訂） | 🟡 |
| O6 | Service 編輯不能切「啟用 / 停用」（要回列表卡片才有狀態，但其實連狀態切換都沒有 UI） | 🟡 |
| O7 | 行事曆沒「日視圖」（很多教練看 daily 比 weekly 順） | ⚪ |
| O8 | 沒法 ban / block 麻煩客戶 | ⚪ |

## 👨‍🏫 Staff（助教）

| # | 問題 | 嚴重度 |
|---|------|--------|
| S1 | Services 頁沒明示是唯讀（看起來像可以新增/編輯，但其實不行） | 🟡 |
| S2 | Sidebar 跟 Owner 看起來一樣，沒看到「助教管理」就知道自己沒權限，但對方無「我是 staff」明顯標示 | ⚪ |

## 👤 Customer 學員

| # | 問題 | 嚴重度 |
|---|------|--------|
| C1 | 公開頁只能看 7 天 — 學員想預約下個月就需要往後 scroll，但沒這 UI | ⭐ |
| C2 | 公開頁服務 description 沒顯示（我設了 DB 欄位但 UI 略過了） | ⭐ |
| C3 | 取消預約後無法**改期** — 只能再從頭預約另一個時段 | 🟡 |
| C4 | 沒辦法看過去預約過哪些教練（無「歷史清單」總覽） | ⚪ |
| C5 | 公開頁無教練聯絡方式（暫時用 invite 系統替代） | ⚪ |

## 🌐 跨角色 / 系統面

| # | 問題 | 嚴重度 |
|---|------|--------|
| X1 | 手機版排版沒做 responsive 測試（雖然 Tailwind 預設不錯，但 sidebar 等沒處理） | 🟡 |
| X2 | 取消 / 刪除動作無確認 dialog（直接 `confirm()`，很陽春） | ⚪ |
| X3 | 沒有 toast 持久顯示成功操作（每次都即時消失） | ⚪ |

# 🚀 本次要修的 ⭐ 項目

1. **O1**：Owner 的行事曆支援「全部成員」+「個別篩選」（用 checkbox 多選）
2. **O2**：新增「重複規則」管理頁 — 列表、編輯、刪除（含級聯刪除生成的 slots 選項）
3. **O3**：行事曆上的時段點擊 → 跳出 detail panel（誰預約了、何時建立的）+ 刪除按鈕
4. **O4**：新增「學員」頁 — 列出有跟自己預約過的所有客戶 + 預約次數
5. **C1**：公開頁從 7 天延伸到 14 天 + 加「下一週」按鈕
6. **C2**：公開頁顯示服務 description
7. **P1**：平台管理員可從租戶列表直接 impersonate-view（唯讀進入該租戶後台）

修完後再來決定下一輪。
