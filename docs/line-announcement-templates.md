# LINE 公告文案範本

租戶 owner 可直接複製貼到 LINE 群組 / 官方帳號公告的文案。`{N}` 換成該服務實際的
`cancel_deadline_hours`（預設 24）；多數服務若都是 24 小時，直接寫「24 小時」即可。

文案用詞刻意與 App 內畫面一致：
- 取消政策框 — `src/app/book/[slotId]/page.tsx`（「開始前 N 小時以上取消，堂數退回套裝；逾時取消或未到場視同已上完一堂」）
- `/my-bookings` 狀態說明 — 「未到場 · 未退還此堂」（系統 `no_show` 標籤，cron 課後自動標記）

## 預約與堂數規則（2026-06-13 更新）

```
【預約與堂數規則更新】

為了讓時段安排更順暢，堂數的計算方式調整如下：

📌 扣堂時機
　預約成功當下即扣 1 堂（不是上完課才扣）。

✅ 可退堂（開課前 {N} 小時以上取消）
　在開課前 {N} 小時或更早自行取消 → 該堂退回套裝，時段也會釋出給其他同學。

⛔️ 不退堂
　• 開課前 {N} 小時內才取消 → 仍可取消、釋出時段，但該堂不退。
　• 當天沒到場 / 沒簽到 → 系統標記「未到場」，該堂不退。

🙋 特殊狀況
　由教練／管理者代為取消（如臨時調課、團班人數不足）→ 一律退堂，不受時間限制。

有任何問題歡迎直接在群組詢問 🙏
```
